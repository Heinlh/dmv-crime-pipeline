What this is

This platform pulls incident-level crime data directly from Montgomery County (dataMontgomery Socrata API) and the District of Columbia (MPD FeatureServer), standardizes it into a shared taxonomy, and republishes it as an interactive map and analytics dashboard. Data reflects what each agency has published as of the last pipeline run, not real-time activity, and locations stay at the block level as published by the source agencies.

See Live Map for recent incidents, Trends for volume and category breakdowns, and About the Pipeline for the architecture behind it.

Data as of last published agency data, not live. Sources: dataMontgomery Crime dataset (icn6-v9z3), DC MPD Crime Incidents feature layers.
