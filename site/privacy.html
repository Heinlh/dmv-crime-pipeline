Privacy // plain-English policy

Privacy Policy

Last updated July 3, 2026.

The short version

  • This site shows public crime data published by government agencies. It does not collect accounts, set its own cookies, or run analytics.
  • If you sign up for email alerts, your address goes directly to Buttondown, our email provider; this site and its public code repository never store it.
  • Like every website, the services that host and serve this site's assets see standard web request data such as your IP address.

Data we collect ourselves

None. DMV Crime Watch is a static website: there is no login, no form that submits to us, no tracking pixel, no analytics script, and no cookies set by our code. We cannot see who visits the site.

Third-party services this site uses

Serving any website requires infrastructure, and those providers receive the standard technical data of a web request (your IP address, browser user agent, and the URL requested) when your browser loads their resources:

  • GitHub Pages hosts the site (GitHub privacy statement). GitHub may log visitor IP addresses for security.
  • CARTO / OpenStreetMap serve the map tiles on the Map page (CARTO privacy, OSMF privacy).
  • Google Fonts serves the heading typeface (Google privacy).
  • unpkg and jsDelivr serve the open-source map and charting libraries.

We chose these providers for delivery only; we receive no visitor data from them.

Email alerts

  • The signup form on the Alerts page submits your email address directly to Buttondown (their privacy policy), which stores the subscriber list and sends the daily brief.
  • Subscription uses double opt-in: you must click a confirmation link before receiving anything.
  • Every email contains an unsubscribe link; unsubscribing stops all email immediately.
  • Your address is used only to send the daily brief. It is never sold, shared, or used for anything else, and it never appears in this project's code repository or on this site.
  • The site owner can see the subscriber list and aggregate signup counts in Buttondown's dashboard, as any newsletter operator can.

About the crime data itself

  • All incident data shown here is already public, published by the DC Metropolitan Police Department, Montgomery County, Prince George's County, and the Fairfax County Police Department through their official open data programs.
  • The agencies anonymize locations to the block level before publishing; this site never sharpens, re-identifies, or augments locations, and displays no names or personal details -- the source data contains none.
  • If you believe a published record is inaccurate, the correction has to happen at the source agency; this site refreshes from the agencies daily and will pick up their corrections automatically.
